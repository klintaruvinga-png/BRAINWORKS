## Purpose

BrainWorks.md is the persistent cognitive model of Kudzie.

It is not a diary. It is not a conversation log. It is not a project record.

It is an evolving model of how Kudzie thinks - principles, patterns, preferences, blind spots, skills, goals, and recurring behaviours. Every AI agent working with Kudzie reads this before beginning significant work. Every addition must improve future reasoning.

`BrainWorks.md` is curated. Producers do not write here directly. OpenClaw curates this file from validated evidence in `observations.jsonl`.

| Rating | Meaning |

## Owner Sections

These sections evolve from observed session data and the rule-of-three promotion protocol. Kudzie influences the model only through session input that is observed and reported. There is no approval gate, and direct decisions about what the model should say do not count as evidence on their own.

OpenClaw is the curator responsible for promotion, conflict handling, and changelog updates. All other agents are evidence producers only.
